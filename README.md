# Glimmer Accessories Website

Simple landing page for Glimmer Accessories with Arabic content, product showcases, and contact links.

## Files
- `index.html` — main page structure
- `style.css` — page styling
- `script.js` — page behavior
- `favicon.svg` — site icon
- `images/` — product and social images

## Usage
1. Open `index.html` in your browser.
2. Upload the repository to GitHub.
3. Share the repository link with others.

## GitHub deployment
If you want to publish it as a GitHub Pages site:
1. Create a GitHub repository.
2. Push this project to the repo.
3. In GitHub repo settings, enable GitHub Pages from `main` branch.
